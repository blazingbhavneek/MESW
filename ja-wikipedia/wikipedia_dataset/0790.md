# 分圧回路

分圧回路（ぶんあつかいろ）または分圧器（ぶんあつき）とは電気回路において、印加された電圧を所定の比で分割する回路、または、機器である。
分圧するための素子として抵抗器の他、インダクタやコンデンサを用いる場合もあり、直流回路、交流回路に対して同様に適用できる。

抵抗分割の法則

# 古典制御論

古典制御論（こてんせいぎょろん、英語：classical control theory）は、伝達関数と呼ばれる線形の入出力システムとして表された制御対象を中心に、周波数応答などを評価して望みの挙動を達成する制御理論である。1950年代に体系化された。代表的な成果物と言えるPID制御は、その扱い易さから現在でも産業では主力である。
